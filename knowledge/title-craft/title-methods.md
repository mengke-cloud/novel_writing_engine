# 标题设计方法

## 适用范围

用于书名、卷名和章节名设计。

## 核心原则

标题首先传达类型和内容承诺，其次追求记忆点。章节名不得提前泄露关键反转。

## 执行方法

- 书名：组合核心对象、异常处境、能力或目标，生成不同方向后筛选。
- 卷名：概括本卷不可逆变化或核心矛盾。
- 章节名：抓取本章最有辨识度的物件、选择、台词或结果。

候选标题按 `类型清晰度 / 内容相关度 / 辨识度 / 可读性 / 泄密风险` 评分。

## 常见错误

- 只追求文艺感，看不出类型。
- 所有章节使用同一种四字结构。
- 标题承诺了正文没有的内容。
- 直接公布悬疑答案或人物真实身份。

## 检查清单

- [ ] 标题与实际内容相关。
- [ ] 能让目标读者判断基本体验。
- [ ] 与同项目其他标题有区分。
- [ ] 不依赖生僻字和无意义符号。
- [ ] 没有提前泄露关键答案。
