# 卷纲格式

## 适用范围

用于 `volume` 阶段规划当前卷。

## 核心原则

- 一卷围绕一个可解决或发生质变的核心冲突展开。
- 冲突、情绪和人物关系都应逐级变化。
- 每个阶段都要产生下一阶段必须处理的后果。

## 常见错误

- 卷目标无法判断是否完成。
- 中段重复同等级冲突。
- 高潮没有付出代价或改变关系。
- 用突然出现的新敌人代替自然升级。

## 检查清单

- [ ] 卷首与卷末状态存在明确差异。
- [ ] 冲突至少发生一次性质或等级升级。
- [ ] 章节区间能够覆盖全部卷级节点。
- [ ] 卷末结果自然产生下一卷问题。
