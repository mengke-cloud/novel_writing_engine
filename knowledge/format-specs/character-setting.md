# 人物设定格式

## 适用范围

用于 `setup`、`outline` 阶段生成主要人物设定。

## 核心原则

- 人物设定必须能够解释其选择，而不只是履历。
- 目标、欲望、缺陷和压力必须形成因果关系。
- 初始设定与剧情中已经发生的状态变化分开保存。

## 常见错误

- 用性格形容词代替可观察行为。
- 主角没有会造成实际后果的缺陷。
- 配角只为主角提供功能，没有自己的诉求。
- 人物能力超过世界规则。

## 检查清单

- [ ] 人物目标能推动至少一条情节线。
- [ ] 面临两难时，可以由设定推导其选择。
- [ ] 人物之间存在诉求差异或信息差。
- [ ] 待确认内容没有冒充既定事实。
