# 章节质量检查

## 适用范围

用于正文初稿、改稿和归档前验收。

## 核心原则

先检查事实和人物是否成立，再检查情节效果，最后处理语言。低层润色不能掩盖高层问题。

## 执行方法

按顺序检查：

1. 连续性：时间、地点、物品、伤势、能力、信息知情范围是否与记忆一致。
2. 人物一致性：行动是否由目标、关系、认知和压力共同推动。
3. 情节推进：本章结束后，局势、关系、信息或目标至少一项发生变化。
4. 场景有效性：每个场景都有目标、阻力、结果，并自然触发下一场。
5. 阅读体验：开篇快速建立问题，中段避免重复，章尾提供明确后续承诺。
6. 语言质量：删去解释性复述、空泛总结、同义重复和模板化反应。

发现问题时按 `阻断 / 重要 / 优化` 标记。阻断项必须修复后才能归档。

## 常见错误

- 只检查错别字，不检查事实冲突。
- 人物为了推进剧情突然改变立场。
- 本章热闹但结束后没有任何变化。
- 章尾只截断动作，没有真正的新问题。

## 检查清单

- [ ] 无时间、空间、物品和能力冲突。
- [ ] 关键行动符合人物当前认知。
- [ ] 章纲核心任务已经完成。
- [ ] 至少产生一项不可忽略的变化。
- [ ] 开篇和章尾均具有推进作用。
- [ ] 阻断项为零。
