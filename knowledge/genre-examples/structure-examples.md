# 类型结构案例

## 适用范围

用于大纲和章纲阶段理解类型循环，不用于直接生成正文。

## 核心原则

学习因果结构，不复制人物、设定、措辞和具体事件。

## 可执行方法

### 成长型

错误：主角受辱后突然突破，击败所有人。

有效结构：旧方法失败 -> 发现规则缺口 -> 为尝试付出可见代价 -> 新方法只解决当前问题 -> 胜利暴露更高层限制。

### 都市目标型

错误：主角展示财富，所有质疑者立刻服从。

有效结构：争取现实资格 -> 资源不足迫使结盟 -> 成果改变议价权 -> 旧关系重新站队 -> 新责任限制下一步选择。

### 悬疑调查型

错误：关键证据在结尾首次出现。

有效结构：异常事实出现 -> 常规解释暂时成立 -> 旧证据因新背景改变含义 -> 人物动机暴露 -> 早期细节完成回证。

## 常见错误

- 只替换名称和背景。
- 把案例顺序当作固定模板。
- 忽略项目已有的人物动机。

## 检查清单

- [ ] 新设计拥有不同的目标与限制。
- [ ] 因果链适配当前项目。
- [ ] 结果改变了项目中的真实状态。
- [ ] 没有复用案例的具体表达。
