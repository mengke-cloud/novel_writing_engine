# 扩展类型结构案例

## 适用范围

用于理解言情、历史、科幻、末世、无限流和同人的结构差异，不用于直接生成正文。

## 核心原则

只学习因果结构，每次使用都必须替换人物目标、限制、关系和结果。

## 可执行方法

- 言情：共同目标迫使合作 -> 边界试探 -> 选择暴露价值差异 -> 信任改变 -> 新关系带来新责任。
- 历史：制度限制目标 -> 寻找有限代理人 -> 交换利益 -> 改变局部权力 -> 引发更大连锁反应。
- 科幻：技术解决旧问题 -> 暴露新成本 -> 社会采用产生分层 -> 人物必须选择由谁承担代价。
- 末世：资源缺口出现 -> 群体提出不同方案 -> 行动获得资源但损失信任 -> 新秩序形成。
- 无限流：表面规则保护参与者 -> 异常证据出现 -> 发现规则边界 -> 付出代价验证 -> 利用边界通关。
- 同人：改变原作早期选择 -> 旧事件失去成立条件 -> 原作人物基于核心动机作出新选择 -> 时间线分叉。

## 常见错误

- 仅替换类型名词，保留同一冲突结构。
- 复制示例中的具体事件和结果。
- 忽略项目人物与世界规则。

## 检查清单

- [ ] 案例只作为结构参考。
- [ ] 新情节符合当前类型承诺。
- [ ] 人物选择来自项目内动机。
- [ ] 结果写入项目时间线和承诺台账。
