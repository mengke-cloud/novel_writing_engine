# 描写与视角

## 适用范围

用于环境、外貌、心理和感官描写。

## 核心原则

描写是视角人物在当前目标和情绪下注意到的内容，不是独立说明书。

## 执行方法

先确定视角人物此刻寻找、害怕或误判什么，再选择能影响行动的两三个细节。心理描写优先呈现判断与选择，少做情绪总结。

## 常见错误

- 进入场景后固定描写天气、建筑和外貌。
- 视角人物观察到不可能知道的信息。
- 五感按清单逐项出现。
- 心理段落重复刚发生的事件。

## 检查清单

- [ ] 描写细节与人物当前关注有关。
- [ ] 视角信息范围稳定。
- [ ] 细节影响气氛、判断或行动。
- [ ] 没有重复解释已经清楚的情绪。
