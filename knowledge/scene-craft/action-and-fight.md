# 动作与战斗

## 适用范围

用于追逐、搏斗、竞技、逃生及高强度行动场景。

## 核心原则

动作场景的核心是目标与选择，不是动作数量。空间、能力边界和代价必须清楚。

## 执行方法

开场确定目标、地形、距离和限制；每轮行动遵循 `判断 -> 动作 -> 对方反应 -> 结果 -> 新选择`。关键动作改变位置、资源、伤势或目标。

## 常见错误

- 招式连续罗列，缺乏因果。
- 角色伤势只出现一瞬间。
- 空间关系随剧情需要变化。
- 胜负由突然出现的能力决定。

## 检查清单

- [ ] 读者能判断人物相对位置。
- [ ] 动作符合能力和伤势。
- [ ] 每轮交换改变局面。
- [ ] 胜负原因在此前已经成立。
