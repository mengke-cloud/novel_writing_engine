# 玄幻仙侠指南

## 适用范围

适用于以力量成长、超凡秩序和更大世界展开为主要期待的故事。

## 核心原则

- 成长必须改变主角可解决的问题，而不只是数值上升。
- 力量体系必须有边界、代价和克制关系。
- 世界扩张要带来新秩序与新选择。

## 执行方法

以“受限 -> 发现路径 -> 付出代价 -> 能力兑现 -> 更高层限制”形成成长循环。每次升级至少改变战斗方式、社会位置或认知中的一项。

## 常见错误

- 境界只换名字，解决问题的方法不变。
- 配角只负责震惊和说明主角强大。
- 临战突然获得未铺垫能力。
- 世界扩大后旧关系全部失效。

## 检查清单

- [ ] 当前成长目标具体可判断。
- [ ] 升级存在过程和代价。
- [ ] 强弱关系符合已确认规则。
- [ ] 每个大阶段都兑现一次核心能力。
