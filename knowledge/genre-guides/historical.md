# 历史题材指南

## 适用范围

适用于真实历史背景、架空历史及强调制度与时代约束的故事。

## 核心原则

- 人物选择受时代制度、身份、交通和信息条件约束。
- 历史背景通过行动后果呈现，不写成百科说明。
- 架空改动必须说明影响链，不能只换一个结果。

## 执行方法

为关键事件建立 `制度条件 / 权力主体 / 资源来源 / 信息速度 / 身份代价` 五项约束。真实史实与虚构内容分开记录，无法确认的内容明确标记。

## 常见错误

- 人物直接使用现代价值和现代组织方式解决问题。
- 仿古词堆砌，但制度、物价和交通悬空。
- 改变一个历史节点却不处理后续连锁影响。
- 所有人物都知道宏观历史全貌。

## 检查清单

- [ ] 身份决定了人物可接触的资源和信息。
- [ ] 行程与通讯符合时代条件。
- [ ] 虚构改动的直接后果已进入时间线。
- [ ] 史实、推测和虚构边界清楚。
