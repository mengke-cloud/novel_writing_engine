# 言情题材指南

## 适用范围

适用于以亲密关系建立、变化和选择为主要阅读期待的故事。

## 核心原则

- 感情进展必须由共同经历和具体选择推动。
- 双方都有独立目标、边界和成长空间。
- 外部阻力应放大关系差异，不能长期替代真实沟通。

## 执行方法

按 `注意 -> 试探 -> 建立信任 -> 边界变化 -> 核心冲突 -> 主动选择` 设计关系阶段。每次推进至少改变信任、依赖、权力、秘密或共同目标中的一项。

## 常见错误

- 只用外貌和身体反应证明吸引力。
- 依靠可立即解释的误会反复拉扯。
- 一方没有独立生活，只服务另一方成长。
- 越界行为被叙事自动包装成深情。

## 检查清单

- [ ] 双方都有清晰且可能冲突的目标。
- [ ] 关系变化有事件和行为证据。
- [ ] 亲密升级伴随新的责任或风险。
- [ ] 核心选择符合人物价值排序。
