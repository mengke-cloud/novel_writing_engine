# 悬疑题材指南

## 适用范围

适用于以问题求解、证据解释和真相揭示为主要驱动力的故事。

## 核心原则

- 真相必须能由已展示信息回溯验证。
- 悬念来自信息意义不明，而非作者无理由遮挡。
- 每次调查都要改变假设、风险或行动方向。

## 执行方法

建立 `事实 / 证据 / 当前解释 / 替代解释 / 知情人物` 五列表。揭示新信息时，同时说明它推翻了什么、留下了什么。

## 常见错误

- 侦探凭直觉直接命中答案。
- 结局引入此前不存在的关键人物或规则。
- 红鲱鱼与真相毫无关系。
- 角色知道真相却只为拖延而不说。

## 检查清单

- [ ] 核心谜题可一句话表述。
- [ ] 关键答案至少有两处前置依据。
- [ ] 每条误导都有合理来源。
- [ ] 揭晓同时解决事实和人物动机问题。
