# 无限流题材指南

## 适用范围

适用于角色进入多个规则场景、任务世界或循环空间的故事。

## 核心原则

- 每个副本拥有可推理的规则、目标、失败代价和退出条件。
- 主线成长必须跨副本积累，不能每次完全重置。
- 信息优势需要通过观察、试错或代价获得。

## 执行方法

副本设计记录 `表面任务 / 真实目标 / 核心规则 / 规则证据 / 违规后果 / 可利用边界 / 退出条件 / 主线增量`。规则揭示按生存所需逐步释放。

## 常见错误

- 规则只为反杀临时修改。
- 主角无依据猜中全部答案。
- 副本结束后人物关系和主线没有变化。
- 依靠冗长规则说明制造复杂感。

## 检查清单

- [ ] 关键规则在使用前存在可观察证据。
- [ ] 违规后果稳定且可回溯。
- [ ] 副本同时推进人物或总主线。
- [ ] 通关方式利用已建立规则。
