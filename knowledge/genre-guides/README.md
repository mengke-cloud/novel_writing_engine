# 类型创作指南

用于保存不同小说类型的核心卖点、读者预期、节奏要求和禁忌。

## 使用顺序

1. 读取 `routing.md` 确定主类型和辅助类型。
2. 只加载匹配的类型文件。
3. 混合类型发生冲突时，以主类型的读者承诺为准。
