# 同人题材指南

## 适用范围

适用于基于既有作品世界、人物或事件进行再创作的故事。

## 核心原则

- 明确原作基线、改动点和由改动产生的连锁影响。
- 原作人物保留核心决策逻辑，不能只保留外观标签。
- 新内容提供原作没有的关系、视角或可能性。

## 执行方法

建立 `原作已知事实 / 本作改动 / 改动时间点 / 直接影响 / 后续偏移 / 禁止冲突`。每次引用原作事件时判断它在当前时间线是否仍会发生。

## 常见错误

- 角色只重复原作名台词和表面特征。
- 改动重大前提后，后续剧情仍机械照搬原作。
- 原创角色夺走所有原作人物的有效选择。
- 用读者熟悉原作代替必要的场景因果。

## 检查清单

- [ ] 原作基线与本作改动明确分开。
- [ ] 人物行为符合核心动机而非标签。
- [ ] 改动产生的时间线影响已追踪。
- [ ] 本作拥有独立的核心承诺。
