# 末世题材指南

## 适用范围

适用于灾变、生存秩序崩塌、资源重组和群体选择为核心的故事。

## 核心原则

- 生存压力来自资源、环境、信息和组织问题的组合。
- 每次获得安全都应产生新的维护成本。
- 新秩序由利益、能力与信任形成，不靠口头宣布。

## 执行方法

持续追踪 `食物 / 水 / 能源 / 医疗 / 住所 / 交通 / 信息 / 人力`。阶段循环采用 `威胁出现 -> 资源取舍 -> 群体分歧 -> 行动代价 -> 新秩序或新缺口`。

## 常见错误

- 资源只在需要制造冲突时才短缺。
- 主角无限储备消除全部生存压力。
- 群体角色只分为忠诚者和恶人。
- 灾变规模宏大，但日常生活不受影响。

## 检查清单

- [ ] 关键资源数量和来源可追踪。
- [ ] 伤势、疲劳和路程产生持续影响。
- [ ] 群体决策存在不同利益。
- [ ] 安全区具有维护成本和内部矛盾。
