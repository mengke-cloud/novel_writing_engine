# 科幻题材指南

## 适用范围

适用于以技术、科学假设及其社会或人物后果为核心驱动力的故事。

## 核心原则

- 核心假设必须改变人物能做什么、不能做什么。
- 技术能力、成本、失败模式和扩散影响同时成立。
- 科学说明只提供理解当前选择所需的信息。

## 执行方法

先写一句核心假设，再列出 `能力 / 限制 / 成本 / 误用 / 社会影响 / 人物代价`。剧情冲突优先来自假设产生的新选择，而不是给旧故事更换科技外壳。

## 常见错误

- 技术在关键时刻拥有临时新增功能。
- 专业术语代替因果解释。
- 全社会面对颠覆技术却保持原样。
- 人物只负责讨论概念，没有个人代价。

## 检查清单

- [ ] 核心假设可用一句话说明。
- [ ] 技术边界在解决问题前已经建立。
- [ ] 世界对技术产生了制度或生活反应。
- [ ] 高概念最终落到人物选择和后果。
