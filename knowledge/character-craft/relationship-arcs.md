# 人物关系弧光

## 适用范围

用于设计爱情、友情、亲情、同盟、竞争和敌对关系。

## 核心原则

关系变化不是好感数值变化，而是信任、权力、依赖、债务、秘密或共同目标发生改变。

## 执行方法

记录关系的 `表面定义 / 双方真实诉求 / 权力差 / 未说出口的信息 / 当前边界`。每个关键场景只推动一到两项变化，并留下行为证据。

## 常见错误

- 一次谈话解决长期结构性矛盾。
- 配角关系只围绕主角单向变化。
- 所有角色表达亲密或敌意的方式相同。
- 冲突后关系自动恢复原状。

## 检查清单

- [ ] 双方都有独立诉求。
- [ ] 关系变化由具体事件触发。
- [ ] 新关系状态影响后续选择。
- [ ] 群像角色具有不同的连接方式。
