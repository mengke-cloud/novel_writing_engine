# 人物决策引擎

## 适用范围

用于人物设计、章纲推演和人物一致性检查。

## 核心原则

人物选择由 `目标 + 认知 + 价值排序 + 可用资源 + 当前压力` 决定。所谓性格必须能转化为选择差异。

## 执行方法

关键节点依次回答：

1. 此刻最想得到什么？
2. 他认为局势是什么？
3. 哪种损失最不能接受？
4. 他手上有哪些真实选项？
5. 选择暴露了什么，又造成什么代价？

## 常见错误

- 用“善良、冷酷、聪明”直接代替选择过程。
- 人物掌握了自己不可能知道的信息。
- 角色只选择对主线最方便的方案。
- 缺陷只存在于介绍中，从不造成后果。

## 检查清单

- [ ] 关键选择至少存在两个有代价的选项。
- [ ] 选择符合人物当前知情范围。
- [ ] 缺陷或价值排序实际影响了结果。
- [ ] 选择产生后续可追踪的变化。
